# # Builder icons

## Requirements

- node >= 16
- npm >= 7

## Installation

Run `npm install` inside root folder.

### Available Scripts

In the project directory, you can run:

#### `npm run docs:dev`

Runs the documentation.

#### `npm run build`

Builds the app for development to the `build` folder.

#### `npm run build:prod`

Builds the app for production to the `build` folder.
